# PTC GraphQL API

PTC provides industrial IoT and digital manufacturing solutions. The ThingWorx IoT platform API covers device connectivity, thing templates and shapes, property management, services, events, streams, value streams, data tables, InfoTable, and manufacturing analytics for industrial digital transformation.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from PTC REST API
**Documentation:** https://support.ptc.com/help/thingworx/platform/r9/en/ThingWorx/api_documentation.html
